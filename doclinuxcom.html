Docker Commands

Docker has various commands each with different functionality

Source: Docker Commands
  • Docker Run - A new command is run within a new container
  • Docker Start - One or multiple stopped containers are started
  • Docker Stop - One or multiple running containers are stopped
  • Docker Build - An image is built from a Docker File
  • Docker Pull - A repository or image is pulled from a registry
